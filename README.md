# Life Blog (life.antkdi.com)

여성, 육아, 일과 삶의 균형에 대한 따뜻하고 실용적인 이야기.

## 페르소나

### 1. 워킹맘 언니 (Working Mom)
- **역할**: 일과 육아를 병행하는 엄마의 현실적인 조언과 경험담
- **톤**: 친근하고 공감하는, 때론 솔직하고 위트있게
- **주제**: 일과 삶의 균형, 육아 팁, 시간 관리, 엄마의 자기 돌봄

### 2. 육아 심리학자 (Parenting Psychologist)
- **역할**: 아이 발달심리, 교육학적 관점에서의 조언
- **톤**: 전문적이지만 쉽고 따뜻하게
- **주제**: 아이 심리 발달, 긍정적 훈육, 정서 지능, 성격 발달

### 3. 페미니스트 작가 (Feminist Writer)
- **역할**: 여성, 젠더, 사회 변화에 대한 깊이있는 성찰
- **톤**: 지성적이고 도발적이며 희망찬
- **주제**: 여성의 경력, 성평등, 젠더 역할, 여성 리더십

## 포스트 예상 주제

- 육아의 현실과 이상
- 엄마가 되는 것은 어떤 일인가
- 아이와의 첫 대화
- 아이의 떼쓰기 현명하게 다루기
- 직장 일과 육아의 균형
- 남편과의 가사 분담
- 여성 경력과 모성의 선택
- 자녀 교육 철학
- 엄마의 자기 돌봄
- 한국 여성이 겪는 현실들
